# Firebase + React Starter Template

Boilerplate for spinning up a new FenriX game project. Copy this folder into `games/your-game-name/` and start building.

## Setup
1. Copy this folder: `cp -r templates/firebase-react-starter games/your-game-name`
2. `cd games/your-game-name`
3. `npm install`
4. Update `.firebaserc` with your Firebase project ID
5. `firebase login`
6. `npm start` for local dev
7. `firebase deploy` to ship
