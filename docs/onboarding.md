# Onboarding

## Setup Steps

1. Get added to the fenrix-ai GitHub org (contact frenzel.tim1@gmail.com)
2. Clone the repo: `git clone https://github.com/fenrix-ai/FenriX.git`
3. Install Node.js (v18+), npm, and Firebase CLI (`npm install -g firebase-tools`)
4. Set up your AI tools (Claude Pro or Cursor recommended)
5. Read `docs/git-workflow.md` and `docs/ai-workflow.md`
6. Join the Slack channels
7. Introduce yourself and your role in the team channel
