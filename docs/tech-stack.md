# Tech Stack

## Frontend
- React (via Firebase Hosting)
- Responsive design for desktop, tablet, and mobile browsers

## Backend
- Google Firebase (Firestore for real-time database, Firebase Auth for login)
- Cloud Functions for server-side game logic and scoring

## Data and Analytics
- Python (pandas, scikit-learn) for dataset generation and model validation
- CSV export for player-side model building

## AI Development Tools
- Claude (architecture, code review, complex logic)
- Cursor (rapid code generation and iteration)
- Codex / Copilot (inline completions)
- Gemini, DeepSeek, or others as needed for specific tasks (image generation, etc.)

## Version Control
- GitHub (fenrix-ai org)
- See `git-workflow.md` for branching and PR conventions
