# Game Design Proposal Template

## Game Title
Working title for the game.

## Concept (2-3 sentences)
What is the game? What do players do? How do they win?

## Target Variable
What are players predicting? (Must be continuous or integer, not categorical.)

## Round Structure
How many rounds? What happens each round? How long per round?

## Player Decisions
What choices can players make each round? List all decision inputs.

## Scoring System
How is the winner determined? What metric is tracked?

## Competitive Dynamics
How do players interact with each other? What prevents a dominant strategy?

## Data Requirements
What does the dataset need to contain? What columns and structure?

## MVP Scope
What is the minimum set of features needed to play one complete game session?
